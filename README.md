# 📝 Paste App

## Overview
Paste App is a fast and simple paste management web application built using **React, Redux Toolkit, and Tailwind CSS**.  
It allows users to create, edit, search, copy, share, and delete text snippets easily with a clean UI experience.

---

## Features
- Create and save text snippets (pastes)
- Edit existing pastes
- Search through saved pastes
- Copy paste content with one click
- Share pastes via link
- Delete unwanted pastes
- Clean and responsive UI

---

## Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS
- JavaScript (ES6+)
- Local Storage
